# How to improve this portfolio referencing.


## Sphinx sitemap

Add ```sphinx-sitemap``` to ```requirements.txt```

Edit ```conf.py```

```python
# -- Referencing --
extensions += ['sphinx_sitemap']
html_baseurl = 'https://vferat.github.io'
html_extra_path += ["extra"]
```
